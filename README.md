# (PoC) openai-realtime-rs

## OpenAI Realtime API Rust Client Library (Proof of Concept)

## Install

```toml
[dependencies]
openai-realtime = { git = "https://github.com/katya4oyu/openai-realtime-rs.git" }
```

## Examples

### Audio to Audio

```bash
cargo run --example a2a

```
